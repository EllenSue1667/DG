Quiz for Lessons 1-5

Parts of Speech - Verbs

Instructions: List the verbs or verb phrases in the sentences, and tell whether they are action verbs or state of being verbs. For extra credit, find the helping verbs.

1. Jim plays basketball.

Jim plays basketball.  -- action verb

2. They will return on the airplane.

They will return on the airplane.  -- action verb
    will - helping verb
    return - main verb

3. Badger is a funny dog.

Badger is a funny dog.  -- linking or state of being verb

4. I have been here a long time.

I have been here a long time.  -- state of being verbs
    have - helping verb
    been - main verb

5. I should have been playing the drum.

I should have been playing the drum.  -- action verbs
    should have been - helping verbs
    playing - main verb

6. Go home.

Go home.  -- action verb

Hint: The last verb in a verb phrase is always the main verb.
